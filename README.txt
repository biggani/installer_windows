README

Run install64.bat as Administrator to install Entermedia
The default install directory is C:\Entermedia.

After the script finishes, open up a web browser and go to http://localhost:8080/

For Document Conversion support (e.g. .doc, .docx), Download LibreOffice:

1. Go to http://www.libreoffice.org/download/libreoffice-fresh/
2. Click the Download button to download the latest .msi
3. Run the .msi and go through installation process
4. Add directory of soffice.exe to %PATH% variable (e.g. C:\LibreOffice\program)
5. Verify that soffice.exe is in the path by opening up a CMD window and typing "soffice --version" and hit ENTER
